# Pars Protocol

Decentralized governance and treasury protocol built on Lux Network. 10 specialized committees, transparent funding, community-driven decisions.

## Overview

Pars Protocol provides governance infrastructure for the Persian diaspora through:

| Component | Description |
|-----------|-------------|
| [Governance](/docs/governance/council) | Council, Committees, Voting, Resolutions |
| [Tokens](/docs/tokens/pars) | PARS, xPARS, vePARS, MIGA |
| [Treasury](/docs/treasury/overview) | Multi-sig vault, committee allocations |
| [Bonds](/docs/bonds/overview) | OHM-style bonding mechanism |

## Network

| Property | Value |
|----------|-------|
| Chain ID | 6133 |
| Block Time | ~400ms |
| Consensus | Snow++ (Lux-derived) |
| VM | Full EVM compatibility |
| Gas Token | PARS |

## Token System

```
PARS    → Base governance token
xPARS   → Staked PARS with yield sharing
vePARS  → Vote-escrowed for governance weight
MIGA    → Freedom of Information DAO token
```

## 10 Committees

Each committee manages a specific domain with dedicated treasury allocation:

| Committee | Persian | Focus |
|-----------|---------|-------|
| AMN | امن | Security & Infrastructure |
| KHAZ | خزانه | Treasury & Finance |
| DAD | داد | Legal & Justice |
| SAL | سلامت | Health & Welfare |


## Precompiles

Pars Network includes native EVM precompiles for high-performance operations:

| Category | Precompiles | Description |
|----------|-------------|-------------|
| [Cryptography](/docs/crypto) | ML-DSA, ML-KEM, SLH-DSA, FROST | Post-quantum and threshold signatures |
| [DEX](/docs/dex) | PoolManager, Hooks, SwapRouter | Uniswap v4-style DEX operations |
| [FHE](/docs/fhe) | Encrypted operations | Fully Homomorphic Encryption |
| [Oracle](/docs/oracle) | Price feeds, GraphQL | Native oracle and query interface |
| [AI](/docs/ai) | GPU attestation, mining | AI compute verification |
